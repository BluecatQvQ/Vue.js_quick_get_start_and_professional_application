# 2.3.1 前后端渲染对比

前端渲染的优点：

+ 业务分离，后端只需要提供数据接口，前端在开发时也不需要部署对应的后端环境，通过一些代理服务器工具就能远程获取后端数据进行开发，能够提升开发效率。

+ 计算量转移，原本需要后端渲染的任务转移给了前端，减轻了服务器的压力。

后端渲染的优点：

+ 对搜索引擎友好。

+ 首页加载时间短，后端渲染加载完成后就直接显示HTML，但前端渲染在加载完成后还需要有段js渲染的时间。
